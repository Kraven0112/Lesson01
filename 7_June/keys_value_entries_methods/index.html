keys(),values(),entries() methods are for object in javascript.

These methods do not need callback function.

These methods are used in object if using for_of loop

Object.keys(object_name) method is for object in javascript.

It is used to access all the keys from object

Object.values(object_name) method is for object in javascript

It is used to access the values of keys from object in javascript.

Object.entries(object_name) method is for object in javascript

It is used to access the keys and values from object in the form of an array in javascript.
