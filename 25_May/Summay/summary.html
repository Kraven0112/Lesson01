About JSON (Java Script Object Notation)
  • It is lightweight data inexchange format.
  • It is used to change string into Javascript object.
  • It is also used to change object into Javascript string.
  • In JSON parse() and stringify() methods are most important methods for data interexchange.
About Object.seal() and Object.freeze()
  • Object.seal() method is used to prevent an object from addition of new properties and deletion of existed properties from an object. But it allows to modify existed properties, key and values.
  • Object.freeze() methos is used to protect an object from modification,deletion of existed properties and addition of new properties. It means this method doesn not allow any changes after frozen of an object.
