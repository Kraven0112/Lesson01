Accessing siblings, parents, etc using javascript.

.nextElementSibling -- is used to show next element using javascript.

.previousElementSibling -- is used to show the element before that element using javascript.

.nextSibling -- shows the next elemet's text or value. It is text node.

.previousSibling -- shows the previou elemnt's text or value. It is also text node.

console.dir() -- is used to show the object type properties.
