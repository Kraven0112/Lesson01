append and appendChild

append vs appendChild

append () is used to add anything or element in the last of the file.

append() returns undefined.

.cloneNode(true) is used to clone the element and its text anywhere.

appendChild() is also used to add aything or element

appendChild returns that element or node_element which it has appeneded.

without cloneNode() we can append multiple element using append() method.

But we cannot append multiple element by using appendChild() without using cloneNode().
