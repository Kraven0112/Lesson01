localStorage

localStorage is used to store data for particular website.

.getItem("item_name") is used to access item from localStorage.

.setItem("item_name",value) is used to create ietm in localStorage.

JSON.parse() is used to convert stringify into object.

JSON.stringify() is used to convert object into special string

localStorage.clear() is used to remove all data from localStorage.

localStorage.removeItem("item_name") is used to remove single data from localStorage.
